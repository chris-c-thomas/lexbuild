## Decisions Made

- **tiktoken**: Deferred. Keep char/4 heuristic for token estimation. tiktoken may be added later as an opt-in flag for precise counting.
- **`law2md index` command**: Skipped. _meta.json is generated during convert; standalone rebuild is a niche use case.
- **Output READMEs**: Title-level only. Provides a browsable TOC without excessive file bloat. Chapter-level READMEs skipped.
- **Changesets**: Already configured (Phase 2). Root CHANGELOG.md is manually maintained; per-package changelogs auto-generated by `pnpm version-packages`.
