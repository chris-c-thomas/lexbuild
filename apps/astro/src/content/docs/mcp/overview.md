# MCP Server

The LexBuild MCP server is coming soon. It will expose the U.S. Code, the eCFR, the Federal Register, and other official legal texts as resources and tools that AI assistants can query directly through the [Model Context Protocol](https://modelcontextprotocol.io/).

Follow the project on [GitHub](https://github.com/chris-c-thomas/LexBuild).
